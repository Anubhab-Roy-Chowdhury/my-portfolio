## üöÄ Setup/Usage

### Local Development
1. Clone the repository:
   ```bash
   git clone https://github.com/Anubhab-Roy-Chowdhury/my-portfolio.git
   cd my-portfolio
   ```

2. Open the project:
   - If using HTML/CSS/JS: Simply open `index.html` in your browser
   - If using a framework: Install dependencies and follow framework-specific instructions

3. For live server (VS Code):
   - Install the "Live Server" extension
   - Right-click on `index.html` and select "Open with Live Server"

### Deployment
- Can be deployed on GitHub Pages, Netlify, Vercel, or any static hosting service
- For GitHub Pages: Go to repository settings > Pages > Select source branch
